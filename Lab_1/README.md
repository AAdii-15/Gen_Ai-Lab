This project performs image classification using deep learning models.

## Models Implemented

### 1. ResNet-50 (Pretrained)
- Pretrained on ImageNet
- Used for proxy cat-detection accuracy

### 2. Custom CNN
- Built and trained from scratch
- Used for cat breed classification

## Files in this Repository
- resnet_accuracy.ipynb – ResNet inference and accuracy calculation
- custom_cnn.ipynb – CNN training and evaluation
- results.md – Summary of results
- accuracy.txt – Final accuracy values
